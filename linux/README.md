# Модуль: Linux для DevOps

## 🐧 Цель модуля
Освоение навыков работы с Linux-системами, необходимых для DevOps-инженера: работа в командной строке, управление процессами, сетевые настройки, безопасность и автоматизация.

## 📂 Структура
linux/
├── command-line/ # Основы командной строки
│ ├── basics/ # Базовые команды
│ ├── files-directories/ # Работа с файлами и каталогами
│ └── text-processing/ # Обработка текста
│
├── system-administration/ # Системное администрирование
│ ├── users-groups/ # Пользователи и группы
│ ├── permissions/ # Права доступа
│ ├── processes/ # Управление процессами
│ └── services/ # Системные службы
│
├── networking/ # Сетевая настройка
│ ├── network-config/ # Конфигурация сети
│ ├── firewall/ # Настройка фаервола
│ └── troubleshooting/ # Диагностика проблем
│
├── scripting/ # Bash-скриптинг
│ ├── basics/ # Основы скриптов
│ ├── automation/ # Автоматизация задач
│ └── best-practices/ # Лучшие практики
│
└── security/ # Безопасность

## 🎯 Осваиваемые навыки
- Работа в терминале и Bash-скриптинг
- Управление пользователями и правами доступа
- Конфигурация сетевых параметров
- Мониторинг и управление процессами
- Настройка системных служб
- Основы безопасности Linux-систем

## 📚 Практические задания
Каждый подкаталог содержит:
- `task.md` - описание задания
- `solution.md` - мое решение (если требуется текст)
- `scripts/` - скрипты (если требуется код)
- `screenshots/` - скриншоты выполнения (при необходимости)
