
Практика — Создание и использование cgroup с ограничением ресурсов
В этой практике вам нужно создать исполняемый Bash-скрипт cgroup_limit.sh, который выполнит следующие действия:

Создаст новую cgroup с названием test_group.
Установит для неё ограничение памяти — 100 МБ.
Запустит в фоновом режиме процесс sleep 200.
Добавит запущенный процесс в cgroup test_group.
Выведет на экран текущий объём используемой памяти для созданной cgroup.
Требования:

Используйте директорию /sys/fs/cgroup для работы с cgroups.
Процесс должен быть явно добавлен в созданную группу.
Вывод должен строго соответствовать формату ниже.
Пример работы скрипта:

sudo ./cgroup_limit.sh Создана cgroup test_group с лимитом 100МБ памяти Запущен процесс с PID: 4321 и добавлен в cgroup Текущее использование памяти cgroup: 4096 байт

                  
Значение памяти может немного отличаться.

Важно: Запускайте скрипт с правами суперпользователя (sudo).

Не забудьте сделать скрипт исполняемым:

chmod +x cgroup_limit.sh
