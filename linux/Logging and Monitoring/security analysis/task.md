
Практическое задание 1: Анализ безопасности системы
Создайте исполняемый Bash-скрипт security_analyzer.sh, который проанализирует безопасность системы на основе логов и выполнит следующие действия:

Проверит количество неудачных попыток входа по SSH за последние 24 часа
Выведет топ-5 IP-адресов с наибольшим количеством неудачных попыток
Определит количество успешных входов в систему за последние 24 часа
Проверит наличие критических ошибок в системных логах за последний час
Создаст отчёт в файле /tmp/security_report_$(date +%Y%m%d).txt
Требования:

Используйте journalctl для анализа системных логов
Используйте grep, awk, sort и uniq для обработки данных
Скрипт должен выводить результаты на экран и сохранять в файл
Пример работы скрипта:

./security_analyzer.sh
=== Security Analysis Report ===

1 192.168.1.1

Successful logins (last 24h): 12
Critical errors (last hour): 3
Report saved to: /tmp/security_report_20250716.txt
